# 题目

- 求一个字符串的子串的个数

# 分析

- 子串：串中任意个连续的字符组成的子序列称为该串的字串，包括空串和原字符串
- 包含 1 个字符的子串共 n 个
- 包含 2 个字符的子串共 n-1 个
- 包含 3 个字符的子串共 n-2 个
- 包含 4 个字符的子串共 n-3 个
- ......
- 包含 n 个字符的子串共 1 个
- 空串1个
- 所以一共有：1+2+3+···+n+1个 = n(n + 1) / 2 + 1
